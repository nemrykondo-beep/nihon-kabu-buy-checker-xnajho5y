# 日本株 買う前チェッカー

日本株を買う前に、感情買い・高値掴み・損切りライン未設定をセルフチェックするためのWebアプリです。

## できること

- 現在値、直近高値、25日線、出来高倍率を入力
- 単元未満株 / 100株を選択
- 急騰後、損切りライン未設定、FOMOを確認
- スコア、判定、加点理由、確認すべきリスクを表示
- 広告スペースと免責事項を表示

## こんな人向け

- 日本株を買う前に一度冷静になりたい人
- 高値掴みを減らしたい人
- 損切りラインを決めずに買ってしまいがちな人
- FOMOで飛びつき買いしがちな人

## 免責事項

本サービスは、日本株を買う前に入力内容を整理するためのセルフチェックツールです。
表示されるスコアや判定は、投資助言・売買推奨・利益保証を目的としたものではありません。
